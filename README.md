### 🛠️ 개발 환경 세팅
해당 프로젝트는 nvm으로 node의 버전을 관리합니다.

#### Mac
클론을 받은 후 터미널에 해당 명령어를 입력합니다

<code>curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.5/install.sh | bash </code>

<br>

설치가 완료되면 정상적으로 설치가 완료되었는지 확인차 버전을 확인합니다.

<code>nvm --version</code>

정상적으로 버전이 출력되면 Node.js 버전을 설치합니다.

<code>nvm install 20.9.0</code>

<br>

설치가 완료되면 Node의 버전을 20.9.0으로 적용합니다.

<code>nvm use 20.9.0</code>
