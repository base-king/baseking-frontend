# Welcome to Baseking Project! 👋
Next.js + Nest.js 기반 야구 정보 공유 플랫폼 Baseking 입니다!


## 프로젝트 소개 🖥️ 
야구 경기, 팀 정보 등 관련 데이터를 빠르게 조회하고 나만의 팀을 팔로우 하여 응원할 수 있는 웹 플랫폼입니다.
